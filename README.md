# FlowMind AI Dashboard UI

A premium SaaS dashboard interface designed for a modern AI automation platform.

This project showcases a high-end product dashboard built using clean UI/UX principles, light theme design, and responsive layout.

---

## 🚀 Live Demo

https://huzaifamansoor489-ai.github.io/Dashboard-/
---

## 🧠 Project Overview

FlowMind AI is a conceptual SaaS platform that allows users to create, manage, and monitor AI-powered workflows and automations.

This dashboard represents the core user interface of the platform.

---

## ✨ Features

- Clean and modern light-themed UI
- Sidebar navigation with active states
- Top navigation bar with search and profile menu
- Overview cards with key metrics
- Analytics section with activity chart (mock data)
- Workflow management table
- Recent activity panel
- Workflow builder preview (visual flow)
- Integrations grid (Slack, Gmail, Notion, etc.)
- Settings interface with toggles and account controls
- Fully responsive design (desktop, tablet, mobile)

---

## 🎨 Design Highlights

- Light color palette (white, soft gray, subtle accents)
- Minimal and professional layout
- Card-based UI with soft shadows
- Smooth hover effects and transitions
- Clean typography and spacing
- Inspired by modern SaaS tools like Notion and Stripe

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
